# Collection Checklist ✅

- [ ] Warm up sensors (~30 s)
- [ ] Verify serial output and Data Forwarder connection
- [ ] Confirm scenario setup (source, distance, ventilation)
- [ ] Wait 2–3 s for stabilization before capture
- [ ] Capture 30 × 10 s samples (or required number)
- [ ] Confirm CSV header and column order: `timestamp, smoke, voc, co, flame, temperature, humidity`
- [ ] Inspect files in Edge Impulse Studio; re-collect if <25 valid samples
- [ ] Log metadata and environmental conditions
- [ ] Upload to Edge Impulse and tag with label/scenario

Keep this checklist handy during field collection.
