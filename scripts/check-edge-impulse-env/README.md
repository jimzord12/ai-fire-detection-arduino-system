# Edge Impulse Environment Check

## Description
Scripts to verify that the necessary development environment tools are installed and correctly configured for Edge Impulse and Arduino development.

## Available Scripts
- `check-edge-impulse-env.ps1`: PowerShell script for Windows.
- `check-edge-impulse-env.sh`: Bash script for Linux/macOS.

## Usage (Linux/macOS)
```bash
chmod +x check-edge-impulse-env.sh
./check-edge-impulse-env.sh
```

## Usage (Windows)
Right-click the `.ps1` file and select "Run with PowerShell" or run from a terminal:
```powershell
./check-edge-impulse-env.ps1
```
